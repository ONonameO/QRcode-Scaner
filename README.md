## 📝简介

QRcode Scaner（二维码识别器）是一款功能全面的二维码解析工具，支持识别市面上几乎所有类型的二维码，包括百度网盘、夸克网盘等国内网盘的分享链接。

---

## ✨ 主要功能

- **右键菜单识别****：在任意网页图片上右键，选择"识别二维码"即可快速解析
- **框选网页区域**：拖拽鼠标选择网页中的二维码区域，精准识别
- **上传本地图片**：支持 JPG、PNG、GIF、WebP 等常见图片格式
- **粘贴剪贴板图片**：直接从剪贴板读取图片进行识别
- **多结果展示**：当图片包含多个二维码时，分别列出每个识别结果
- **一键复制**：点击复制按钮快速复制识别结果
- **快捷打开链接**：识别结果为网址时，可直接打开链接

---

## 🔧 使用说明

1. **右键识别**：在网页图片上右键 → 选择"识别二维码"
2. **框选识别**：点击扩展图标 → 点击"框选网页区域" → 拖拽鼠标选择二维码区域
3. **上传图片**：点击扩展图标 → 点击"上传本地图片" → 选择二维码图片
4. **剪贴板识别**：复制二维码图片 → 点击扩展图标 → 点击"粘贴自剪贴板"
